# nagios-plugins-eudat-gitlab

GitLab liveness check

## Usage

NAME
```
    check_gitlab_liveness.sh - Gitlab Liveness
```

optional arguments:
  -h, --help            show this help message and exit
  -d, --debug           debug mode

required arguments:
  -u URL, --url URL     GitLab liveness URL to check
  -t TIMEOUT, --timeout TIMEOUT
